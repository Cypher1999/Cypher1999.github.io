QUÉ ES?

Es un lenguaje de programación de alto nivel derivado de C desarrollado con el fin de orientar dicho lenguaje al manejo de objetos. A lo largo del curso del curso de introducción a la Computación se usará este lenguaje para aprender a programar en base a el paradigma orientado a objetos (POO).

ORIGEN

Fue diseñado por Bjarne Stroustup en 1979 con el fin de extender el lenguaje de programación C , agregando mecanismos que permitieran la manipulación de objetos, lo que convierte a C++ en un lenguaje multiparadigma.

RECURSOS

Durante el curso de introducción a la computación se usará el IDE Eclipse para el desarrollo de código c++.

Aunque no es estríctamente necesario usar Eclipse para programar en c++, un IDE facilita aspectos como la depuración de nuestro código, sin embargo, puedes usar tu editor de código o IDE Favorito . Consigue editores y otras herramientas aquí.

Para desarrollar tu código con Eclipse necesitas:

  • El Ejecutable de Eclipse
  • MinGW (compilador de c++ para windows)
  • Una versión de Java JDK

Te dejo los enlaces directos

- NOTAS:
  • MinGW es un compilador de Windows, usuarios con otros sistemas operativos deberán buscar otra alternativa. Usualmente las distribuciones Linux ya cuentan con un compilador de C y C++.
  • Es posible que como usuario, ya cuentes con una versión de Java. Si ese no es el caso e inicias tu ejecutable de eclipse podrías encontrarte con un error en pantalla, entonces podrás instalar una versión en el enlace.
